# 开发文档

## 1.返回数据格式说明
- **code** *Number*
    - 0 请求成功
    - 1 格式错误 - 数据格式不正确 
    - 2 用户名冲突 - 用户名已存在
    - 3 登录失败 - 用户不存在
    - 4 服务器错误
    - 5 密码错误
    - 6 鉴权失败 - 用户未登录
    - 7 修改失败 - 新旧用户名一样
    - 8 修改失败 - 新旧密码一致
    - 9 上传失败

- **msg** *String*
    字符串. 一般用于描述用户请求状况

- **data** *Object*
    请求需要返回对象数据是，会存放在data
